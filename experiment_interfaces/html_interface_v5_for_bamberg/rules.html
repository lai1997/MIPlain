Rules of the game


First, here are some basics of the game. The Island game is a two-player game.
The two players, Green and Orange, take turns conquering territories.
Any untaken territory can be conquered until the game terminates.
You will play Green.

Conquering a territory provides one or several resources.
Below is the initial world. Select one territory to get some resources.

Each resource belongs to one of the following category:
Island 1, Island 2, Island 3, Forest, Castle, Cornfield, River, Animal.

The player who first succeeds in obtaining three resources of the same category wins the game.

You can press the button at the corner of each board to view the table for resources captured.
Pressing "P1 resources" will show the resources captured by Green so far, then pressing "P2 resources" will show the resources captured by Orange so far. Pressing "Board" will get you back to the current board.
Please press the button on the Initial Board above to see the resources captured.
